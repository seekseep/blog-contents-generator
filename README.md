# OpenAIを使って記事を生成

# 環境

```
% node -v
v20.11.1
% npm -v
10.2.4
```

## OpeanAI API Key

- https://platform.openai.com/signup
- https://platform.openai.com/account/api-keys

# セットアップ

```bash
% npm install
```

## 環境変数の定義

```bash
OPENAI_API_KEY=...
OPENAI_MODEL=gpt-4
```

# 実行

```bash
% npm start
```
